Simple Calculator

This Java project implements a simple calculator that performs basic arithmetic operations (+, -, *, /) on two numbers entered by the user.


import java.util.Scanner;

public class Calculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("Simple Calculator");
        System.out.println("Enter two numbers:");

        double num1 = scanner.nextDouble();
        double num2 = scanner.nextDouble();

        System.out.println("Select operation (+, -, *, /):");
        char operation = scanner.next().charAt(0);

        double result;

        switch (operation) {
            case '+':
                result = num1 + num2;
                break;
            case '-':
                result = num1 - num2;
                break;
            case '*':
                result = num1 * num2;
                break;
            case '/':
                if (num2 != 0)
                    result = num1 / num2;
                else {
                    System.out.println("Error! Division by zero is not allowed.");
                    return;
                }
                break;
            default:
                System.out.println("Error! Invalid operation.");
                return;
        }

        System.out.println("Result: " + result);
    }
}

Word Counter

This Java project counts the number of words in a sentence entered by the user.


import java.util.Scanner;

public class WordCounter {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("Word Counter");
        System.out.println("Enter a sentence:");

        String sentence = scanner.nextLine();

        String[] words = sentence.split("\\s+");

        System.out.println("Number of words in the sentence: " + words.length);
    }
}
